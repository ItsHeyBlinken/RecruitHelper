# Arkansas & Louisiana JUCO / NAIA Softball Athletic Websites

Regional expansion: **smaller schools first** (JUCO → NAIA), then higher divisions later.

**Roadmap:** TX (done) → **AR/LA JUCO+NAIA (this list)** → OK JUCO+NAIA → then D3/D2/D1 in those regions.

Home page stays Texas-first; AR/LA programs are available via search and division filters.

### Notes
- **ASU-Newport:** Softball discontinued effective July 1, 2026 — not included.
- **Loyola University New Orleans:** No varsity softball (intramural only) — not included.
- **Arkansas Baptist College:** Listed on some NAIA directories but no confirmed active varsity softball site — not included.
- URLs should be verified before/after first scrape; update this doc when fixed.
